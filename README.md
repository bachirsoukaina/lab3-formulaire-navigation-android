*Fonctionnalités
Saisie : Nom, Email, Téléphone, Adresse, Ville
Validation des champs obligatoires (Nom, Email)
Navigation entre deux activités (Intent)
Affichage des données saisies
Bouton retour vers le formulaire

Java
Android Studio
XML (UI)

Lancer sur un émulateur ou téléphone

Activity
Intent (putExtra / getStringExtra)
